The Request module holds data about the incoming request by the user. The only reserved parameters are module, controller, action, and format. Any other parameters can be injected into the Request at any point during the application's execution.

Methods

  • setParams(array $params)

    Merge one or more parameters with the current Request.

  • params()

    Get all current parameters from the Request

  • param(string $name, mixed $defaultValue = null)

    Get a single parameter from the Request. If the parameter cannot be found, the optional $defaultValue will be returned.

  • set(string $name, mixed $value)

    Set the value of a single parameter in the Request

  • reset()

    Resets the Request parameters to their default values

  • Helper Methods

    Methods for getting and setting values are provided for the reserved parameters:

    • module
    • controller
    • action
    • format
